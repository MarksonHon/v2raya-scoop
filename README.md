# v2raya-scoop
Scoop bucket of v2rayA for Windows users, powered by project v2rayA and project [FastGit](https://fastgit.org/).

## How to use

Add bucket

```ps1
scoop bucket add v2raya https://github.com/v2rayA/v2raya-scoop
```

Update buckets

```ps1
scoop update
```

Install v2rayA

```ps1
scoop install v2raya
```
